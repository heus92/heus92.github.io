4 months startup working on a sign language translation wristband.

Started the project when in UC Berkeley. Back in Paris the customer interviews showed us this was not the right solution to this problem.

expand

This project was originally borned as the jedi wristband where we wanted to interact with our environment using only gestures. We soon realised that if we wanted to build some solid ventures the gadget could not be the main value proposition. This is when we started focusing on sign language.

Our prototypes were going great (sometimes it’s useful to be a mechanical engineer) when we started digging into the translation process. We had missed something pretty big…face expressions. Sign language is not only a sequence of gestures!! No problem we’ll find a new usecase to our incredible tech. VR, gestures learning, virtual keyboard, and so many more were killed because of some interesting patents belonging to Apple, LG, Sony among others.

We’ll remember that we had something interesting enough for tech giants to deposit a patent.
