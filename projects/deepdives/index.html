We started this project because of the recommandation problem we face. Everybody is always flooded into content (articles, videos, musics…) that follow the same stupid principle: you liked it once, you’ll want more and more and more and more of it.

This is why we decided to build our own version of what we felt was the real recommandation system.

Even though we aborted the mission project, I am still working on it as a mean to learn machine learning and deep learning recommandation systems. The Github repo of the project is public and a demo will soon be available at deepdives.io
